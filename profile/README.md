# T.LY URL Shortener

T.LY is the all-in-one URL shortener and QR code platform built for marketers, developers, creators, and teams.

## Features

- Short links  
- Custom-branded domains  
- QR code generation  
- Click tracking & analytics  
- Smart routing  
- Link rotation
